### Recommendations to Improve Interview Chances

1. **Build a portfolio on GitHub** — 1–2 end-to-end projects (e.g. a Power BI sales dashboard, Python EDA on a Kaggle dataset) dramatically improve callback rates
2. **Sharpen advanced SQL** — practice window functions, CTEs, subqueries on LeetCode/HackerRank; most companies run a SQL screening test in round 1
3. **Learn BigQuery or Looker** — both have free tiers; adding either to your resume closes the biggest gap seen across all roles
4. **Get a recognized certification** — Google Data Analytics (Coursera) or Microsoft PL-300 (Power BI) carry strong weight with Indian enterprise recruiters
5. **Optimize LinkedIn** — set "Open to Work", headline as *"Data Analyst | Python · SQL · Power BI · Tableau"*, and list all tools in your Skills section to surface in recruiter searches
